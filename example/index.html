Programação de Aplicativos Mobile

Desenvolvimento de aplicações mobile escaláveis utilizando React Native. O objetivo é criar capacidades técnicas e socioemocionais num contexto de squad ágil, visando implementar processos críticos de campo com tratamento offline.

⚠️ Atenção: Carga Horária e Ambiente de Testes

As práticas desta Unidade Curricular devem ser exclusivamente executadas e testadas em emuladores (Android Studio / Xcode). Não é permitido o uso de celulares físicos (pessoais ou corporativos) durante as atividades da mesma para garantir a uniformidade tecnológica no laboratório.

🎯 Situação de Aprendizagem

Os alunos atuarão como Desenvolvedores Mobile em uma squad ágil. O projeto visa levar processos baseados em terminais fixos da empresa para dispositivos móveis responsivos e seguros, integrando mapeamento GPS, consumo de RESTful APIs e garantindo tolerância a falhas de conexão.

⚙️ Competências Desenvolvidas

👨‍💻 Capacidades Técnicas

  • Utilizar os frameworks de programação e estruturação do lado do cliente de acordo com Boas Práticas de programação.
  • Reconhecer as características dos dispositivos mobile clientes para uso de leiautes responsivos, design adaptativo e serviços de acessibilidade, localização, notificação e comunicação.
  • Desenvolver aplicativos mobile, considerando os protótipos de interface para dispositivos móveis de acordo com artefatos.
  • Aplicar técnicas e linguagens atuais de acordo com escopo do projeto, considerando o ambiente de Interface de Desenvolvimento - IDE.
  • Aplicar testes de sistema de acordo com as suas especificações técnicas e elaborar planos de execução (roteiro, funcionalidade, ferramenta).
  • Aplicar diferentes métodos e frameworks de testes para validação do software e respectivas normas.
  • Aplicar técnicas e procedimentos de segurança para o desenvolvimento de sistemas.
  • Aplicar metodologias ágeis para otimização do processo de programação.

🤝 Conhecimentos

  • Fundamentos e Arquitetura: MVC, ciclo de vida do App, componentes básicos de UI e diferenças entre Nativos/Híbridos.
  • Layout Adaptativo: Design responsivo para multi-telas, orientação e acessibilidade.
  • Integração Web (APIs): Consumo RESTful (XML/JSON), tratamento assíncrono e autenticação.
  • Sensores e Persistência: Armazenamento local (SQLite), gestão de cache, GPS, map filters, NFC e Push Notifications.
  • Segurança Corporativa: Ameaças/vulnerabilidades (XSS/Injeção), criptografia, verificação de entrada e segurança de dados perante perdas físicas (OAUTH/Authorization).
  • Qualidade e Playstore (Deploy): Táticas de debug em Profiler, testes E2E/Unitários, e o protocolo submissão nas grandes Lojas de Aplicativos.
